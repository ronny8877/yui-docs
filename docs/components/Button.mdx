A Normal button user can interact with.

### Usage


In Yui-button we provide a prebuilt button web component with diffrent styels  you can play around with.
The visual styles and behaviou of the button is controled by passing the attribute to the elements.

```html title="variant danger"
<yui-button variant="danger">Red button </yui-button>
```

We kept our component library as simple as possible to use as to get started all we need to is use the component and even if no 
attribute is passed the default style will be applied.

```html
<yui-button> Styled Button </yui-button>
```
The default styling will be applied for all the attributes if no values are passed.

```html
<yui-button outline> Styled Button </yui-button>
```


```html 
<yui-button loading> Styled Button </yui-button>
```

:::caution
If variant and outline attributes are passed together the outline style will be applied.
:::


```html 
<yui-button variant outline> Styled Button </yui-button>
```

## API


| Attribute | Description | Type | Default | 
| --- | --- | --- | --- |
| variant | Set the solid background color of the butto| `primary` \| `secondary` \| `danger` \| `success`\|`warning`\|`info`\|`light`\|`dark`\|`link`|  primary |  
| outline | Set the background to transparent with outine | `outline-primary` \| `outline-secondary` \| `outline-danger` \| `outline-success`\|`outline-warning`\|`outline-info`\|`outline-light`\|`outline-dark`| outline-primary |  
| disabled | Disabled state of button | - | - |  
| size | Set the Size of button | `sm` \| `md` \| `lg` \| `xl`  | md |  
| rounded | Make the button cornor rounded | - | - |  
| loading | Set the loading states of button | `loading` \| `error` \| `done` \| `inline-loading`\|`inline-error`\|`inline-done`| loading |
